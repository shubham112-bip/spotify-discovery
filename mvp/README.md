# Commute Compass — MVP

AI-native Spotify discovery agent for commute listeners.

**Status:** Not started — build begins Day 7 (23 Jun)

## Concept

User provides context (duration, mood, adventure level) → AI generates an 8-track session with **why this song** explanations → creates Spotify playlist.

## Tech (planned)

- Next.js or Streamlit
- Spotify Web API (OAuth PKCE)
- Claude or OpenAI API
- Deploy: Vercel or Streamlit Cloud

## Why AI vs traditional recsys

Traditional systems optimize similarity and engagement. This MVP adds **moment context + natural language + explainability** — the gap validated in Parts 1–2.
